# Pemilahan-Sampah-Logam-dan-Non-Logam-Organik-dan-Anorganik-Berbasis-Sensor-Proximity-dan-Moinsture
Projek ini saya buat untuk mengatasi permasalahan sampah yang ada di perkotaan dengan sistem pengelolaan sampah otomatis yang memanfaatkan teknologi sensor untuk meningkatkan efisiensi dan akurasi proses pemilahan.
