# Pemilahan-Sampah-Logam-dan-Non-Logam-Organik-dan-Anorganik-Berbasis-Sensor-Proximity-dan-Moinsture
Nama  : Ragil Anam Winarya
Tahun : 2026

Projek ini saya buat untuk mengatasi permasalahan sampah yang ada di perkotaan dengan sistem pengelolaan sampah otomatis dengan memanfaatkan teknologi sensor untuk meningkatkan efisiensi dan akurasi proses pemilahan.
Projek ini dirancang untuk mendeteksi dan mengklasifikasikan jenis sampah secara otomatis berdasarkan karakteristik fisik dan kandungan materialnya.
